# BonziBUDDY for Reborn XP
</div>

BonziBUDDY is a faithful recreation of the iconic 2000s desktop companion, built specifically to live inside the **[Reborn XP](https://xp.quenq.com/)** simulator. 

He sings, he tells terrible jokes, and he wanders around your desktop. Best of all? This version is 100% spyware-free. He's just here for the nostalgia.

---

## Prerequisite
This isn't a standalone app for your real computer. It only runs inside **Reborn XP**, the web-based Windows XP simulator.

### How to Install

1.  Launch **[Reborn XP](https://xp.quenq.com/)**.
2.  Open the **Start Menu** -> **All Programs** -> **App Market**.
3.  Search for **BonziBUDDY** and click **Install**.
4.  Launch him from your Start Menu or the new Desktop shortcut!
